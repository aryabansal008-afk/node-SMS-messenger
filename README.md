# node-SMS-messenger
Real time SMS messaging application built with Node.js, Express, Socket.IO, EJS, and the Vonage SMS API.

## Features

- Send SMS messages using the Vonage SMS API
- Real-time status updates with Socket.IO
- Simple and responsive user interface
- Server-side request handling with Express
- Template rendering using EJS
- Form validation and message processing

## Tech Stack

- Node.js
- Express.js
- EJS
- Socket.IO
- Vonage SMS API
- CSS
- JavaScript
